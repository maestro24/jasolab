## 1. 문제 정의 & 검증 근거 (2026-07-13 실조사)

- "글자수 세기"는 사람인·인크루트·카피킬러가 도구를 상시 유지할 만큼 검증된 대형 수요. 단 헤드 키워드는 사람인(초고DR)이 장악 — **정면 승부 불가, 워크플로우 특화로 우회**.
- 기존 도구들의 공백: 전부 "붙여넣고 세기" 일회용. **취준생의 실제 작업 흐름(여러 회사 × 여러 문항 × 여러 버전)을 관리해주는 도구가 없다.**
- 취준생 = 면접 정장·구두·자소서 도서 등 쿠팡 파트너스 전환 최적 + 취업 카테고리 AdSense 단가 상대적 우위 (검증 에이전트 8개 중 수익화 최고 판정).
- 서버 전송 0 (localStorage) = 민감한 자소서 데이터에 대한 실질적 차별화 셀링포인트.

## 2. 핵심 원칙

1. **도구가 아니라 작업대**: 글자수 세기는 미끼, 본체는 "회사별 문항 관리 + 버전 스냅샷". 재방문·북마크가 목표 지표.
2. **데이터는 사용자 브라우저에만**: 자소서는 민감 정보 — localStorage 저장, 서버 전송 0, 이 사실을 전면에 홍보. 단 **데이터 유실 방지 장치 필수** (JSON 내보내기/가져오기, 자동 스냅샷).
3. **글자수 3종 동시 표시**: 공백 포함/제외·바이트(2byte 규칙)·원고지 — 기업마다 기준이 달라서 생기는 불안을 한 화면에서 해소.
4. **기업 프리셋은 참고 정보**: 채용 요강은 매년 바뀜 — "20XX년 기준, 실제 공고 확인" 고지 필수, 틀려도 사이트 신뢰가 안 죽는 표현으로.
5. **3초 사용성**: 착지 즉시 타이핑 가능. 가입·설정 없음 (다모아툴 원칙 계승).
6. **점진 개선**: 단계마다 시각·성능·느낌 +12%, 매 단계 브라우저 실검증.

## 5. 타겟 & 사용 맥락

- 1차: 신입 공채 취준생 (하반기 9~11월, 상반기 3~4월 피크 — 계절성 강함)
- 2차: 경력 이직자 (경험기술서·자기소개서), 대입 자소서 폐지로 고교생 수요는 없음
- 맥락: 여러 회사에 동시 지원 → 문항 유사(지원동기/성장과정/직무역량) → 기존 답변 재조합 + 글자수 맞추기 반복

## 6. 기능 (MoSCoW)

### Must
- **문항 에디터**: 제목 + 본문, 실시간 카운터 3종(공백 포함/제외, 바이트 — 한글 2byte/영문 1byte 규칙, 원고지 200자 매수), 제한 설정 시 초과분 시각 경고 + 진행 바
- **회사 > 문항 트리**: 지원 회사(폴더) 아래 문항 여러 개, 마감일 입력 → D-day 뱃지·마감순 정렬
- **버전 스냅샷**: 수동 "스냅샷 저장" + 문항당 최근 자동 백업, 목록에서 미리보기·복원 (diff까지는 Won't)
- **기업 프리셋**: 주요 기업 글자수 제한 프리셋 (삼성 700자, 현대차·LG·SK·CJ·은행권 등 ~30사) — data/presets.json, 연도·출처·변경가능 고지
- **localStorage 자동 저장** (입력 debounce) + **JSON 전체 내보내기/가져오기** (기기 이전·백업)
- **답변 라이브러리**: 전체 문항 검색 (키워드 → 내가 쓴 답변들) — 재조합 워크플로우의 핵심
- 복사(서식 없이)·txt 다운로드

### Should
- 단독 SEO 페이지: `/count/` 글자수 세기 (헤드 회피, "자소서 글자수 세기"·"원고지 계산" 롱테일), `/presets/<기업>/` 기업별 랜딩 ~30페이지 (가벼운 pSEO)
- 인쇄 스타일시트 (문항 모아 인쇄)
- 목표 글자수 대비 문장 단위 통계 (문장 수·평균 문장 길이)

### Could
- 맞춤법 검사 외부 링크 (부산대 검사기 등 — 연동 아님, 새 탭 링크만)
- 스토리지 사용량 표시 + 용량 경고

### Won't (Phase 2+)
- 이력서 PDF 빌더 (별도 프로젝트 규모 — 성과 보고 분리 판단), AI 첨삭(API 비용), 클라우드 동기화(서버), 버전 간 diff 뷰

## 7. 페이지 구조

```
/                워크스페이스 (회사/문항 트리 + 에디터) — 앱 본체
/count/          글자수 세기 단독 (SEO 유입 → 워크스페이스 전환 유도)
/presets/        기업별 글자수 프리셋 목록
/presets/<기업>/ 기업별 랜딩 ×~30 ("삼성 자소서 글자수" 롱테일)
/guide/          자소서 글자수 규칙 해설 (바이트 계산법 등, E-E-A-T)
```

## 8. 수익화

1. 쿠팡 파트너스 (기존 배너 체계) — **취준 상품군 명확**: 면접 정장·구두·자소서 도서·증명사진 소품. 에디터 화면엔 광고 금지(집중 작업 영역), /count·프리셋·가이드 페이지만
2. AdSense (커스텀 도메인 후) — 취업 카테고리 단가 중상
